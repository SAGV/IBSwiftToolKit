IBSwiftToolKit contains a lot useful extensions.

You can use CocoaPods to integrate library with your project:
```
 pod 'IBSwiftToolKit', :git => 'https://github.com/ivabra/IBSwiftToolKit'
```

**Foundation**
- NSAttributedString
- NSDateFormatter
- NSObject
- NSRange

**Swift**
- Array
- Int
- String

**CoreGrahics**
- CGAffineTransform
- CGColor
- CGImage
- CGPoint
- CGRect

**UIKit**
- UIAlertViewController
- UIBarButtonItem
- UIButton
- UIDevice
- UISearchBar
- UISegmentControl
- UIView
- UIViewController
